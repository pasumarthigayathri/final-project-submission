# Final Project Submission

This repository contains the required files for my peer‑graded assignment.  
It demonstrates the use of GitHub UI and Git CLI tasks as part of the project.

## Files Included
- LICENSE: Apache 2.0 License file
- README.md: Overview of the project
- CODE_OF_CONDUCT.md: Contributor Covenant code of conduct
- CONTRIBUTING.md: Guidelines for contributing
- simple-interest.sh: Shell script to calculate simple interest

## Usage
To run the simple-interest.sh script:
1. Clone the repository using `git clone <repo-url>`.
2. Open a terminal and navigate to the repo folder.
3. Run the script using `bash simple-interest.sh`.
4. Enter the values for Principal, Time, and Rate when prompted.
